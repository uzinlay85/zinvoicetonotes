Notion Integration

သင်၏မှတ်စုများကို သင်၏ Notion Database ထဲသို့ တိုက်ရိုက်သိမ်းဆည်းပါ။

Notion Key နှင့် Database ID ရယူနည်း လမ်းညွှန်

သင်၏ မှတ်စုများကို Notion သို့ပို့ရန် အောက်ပါအဆင့်များအတိုင်း API Key နှင့် Database ID ကိုရယူပါ။

  1. Integration ပြုလုပ်ခြင်း: Notion's Integrations page သို့သွားပြီး "+ New integration" ကိုနှိပ်ပါ။ နာမည်တစ်ခု (ဥပမာ: ZinNotes App) ပေးပြီး Submit လုပ်ပါ။ ပေါ်လာသော "Internal Integration Token" ကို copy ကူးပါ။ ၎င်းသည်သင်၏ API Key ဖြစ်သည်။
  2. Database တည်ဆောက်ခြင်း: သင်၏ Notion Workspace တွင် Page အသစ်တစ်ခုဖွင့်ပြီး /table ဟုရိုက်ကာ "Table - Full page" ကိုရွေးပါ။ Database ကို နာမည်တစ်ခုပေးပါ (ဥပမာ: ZinNotes)။
  3. Integration ကို Database နှင့် ချိတ်ဆက်ခြင်း: သင်တည်ဆောက်ခဲ့သော Database ၏ ညာဘက်အပေါ်ထောင့်မှ အစက်သုံးစက် (...) ကိုနှိပ်ပါ။ "+ Add connections" ကိုနှိပ်ပြီး သင်အဆင့် (၁) တွင်ပြုလုပ်ခဲ့သော Integration ကိုရွေးချယ်ချိတ်ဆက်ပါ။
  4. Database ID ရယူခြင်း: Browser တွင် သင်၏ Database page ကိုဖွင့်ထားချိန် URL ကိုကြည့်ပါ။ https://www.notion.so/your-workspace/DATABASE_ID?v=... ပုံစံဖြစ်ပါလိမ့်မည်။ your-workspace/ နှင့် ?v= ကြားမှ စာလုံးအရှည်သည် သင်၏ Database ID ဖြစ်သည်။
